### How to
## Karaoke game
- needs timidity installed to play midi with pygame
- python requirements in requirements.txt
- go to the karaoke_game folder, start the venv with the requirements installed, then start with python karaoke.py
-try to hit the notes you can see and hear (NOTE: use headphones or set the speaker volume so low that you can sing louder than the output to score points)
